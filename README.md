# apartment_bot
A discord bot that helps manage the apartment I live in. Commands are tailored to access a spreadsheet of particular format.
## Getting started.
1. Install dependencies\
`npm install`
2. Load/Reload discord commands\
`node deploy-commands.js`
3. Start the app\
`npm start`
